# Charter Borders <Badge type="tip" text="Revamp Content" />

An image was leaked that led to speculations that you might be able to
visualise Charter Borders (a.k.a. their [walls](../mechanics/charters#wall))
using a command.

![charter-borders.png](./charter-borders.png)
<div class="subtitle">The leaked image with the borders.</div>

## Greater Charter Stone
Note that in this image, another thing can be seen which is essentially a very big
sort of wrench-shaped stone. In one of the videos of Content SMP, we can see this
acting as a sort of upgrade to the Charter Stone, providing a lot larger area of coverage.
Whether it has more mechanics than just that is currently unknown.

![greater-charter-stone.png](greater-charter-stone.png)
<div class="subtitle">The Greater Charter Stone in RAT's <a href="https://youtu.be/kNPDNK02m2M?t=2">latest CSMP video</a>.</div>

## Command
In another image, we can see a command, `/charter_unveiling`, that might be linked to this phenomenon. It is
likely ran to toggle between visualised or hidden borders.

![charter-borders-command.png](./charter-borders-command.png)
<div class="subtitle">The Visualise Borders command in game.</div>
