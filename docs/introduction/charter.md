# **Contracts, Covenants and Regrets**

Charter is a mod made by [Arathain](https://modrinth.com/user/Arathain) expanding on “Contract deals”
and claiming land without breaking game immersion. It does this by introducing a handful of items 
and features which you can use to make land and most importantly, *people >:)* “yours”.

## Extra info:

- Charter WILL crash Quilt based servers, so we suggest you to use 
[Essential](http://essential.gg)’s lan connection service to play with friends on the same modpack. The fabric versions of Essential work on Quilt.
- You’ll also need a downgraded version of QFAPI to play without a crash. 
We suggest to use version 4.0.0-beta.30+0.76.0-1.19.2 .

Since this Wiki is currently focused on the current beta, this mod in the end WILL 
be updated and all outdated cards that do not apply anymore to the then 
released mod, will be placed in a “Preview Charter” list. Things will be 
salvaged or changed to fit the new release here and there, though.
